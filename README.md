#  Телеграм-бот магазина электроинструментов

Курсовой проект для университета 2 курс

##  Описание
Telegram-бот для интернет-магазина электроинструментов с базой данных SQLite.

## Функционал
- Каталог товаров с категориями
- Корзина покупок
- Регистрация пользователей
- Админ-панель
- Статистика магазина
